Аннотация @Data используется в библиотеке Lombok в Java и предназначена для автоматического создания стандартных методов класса. При добавлении @Data к классу, Lombok автоматически генерирует:

    Геттеры для всех полей.

    Сеттеры для всех нестатических полей.

    Метод toString().

    Методы equals() и hashCode().

    Конструктор без аргументов (если не определён другой).

Пример использования:

import lombok.Data;

@Data
public class User {
    private String name;
    private int age;
}

Этот код эквивалентен вручную написанному классу с геттерами, сеттерами, toString(), equals(), hashCode() и конструктором по умолчанию.

Важно: Lombok должен быть подключён к проекту, и IDE должна поддерживать его (например, через плагин).
